Как работает программа? (ВСЁ В ПОРЯДКЕ ТОГО, КАК ДОЛЖНА НАЧИНАТЬСЯ РАБОТА) :

- Output folder - Папка, куда будут выгружаться файлы (Выбираете куда кинете файлы, и естественно врубится опция URL, без этого эта строка работать нормально не будет)
- URL - Ссылка на видео или плейлист
- Preset format - Выбор формата файла :
	- Audio - MP3 файлы это музыка, всё автоматом конвертируется.
	- Video - WEBM файлы, если прожмёте галочку с Remux/Convert, то файл будет MP4. 
	- Custom - specify format string. В ней можете указать любой формат файла, который вам нравится??? Ну, не уверен, что прям любой формат скачается, так что да.

Остальные опции не совсем нужные, но поясню, что там тоже есть :
- Download subtitles - Скачивает субтитры, если они имеются.
- Embed Thumbnail - Скачивает и вставляет обложку на видео.
- Remux/Convert after download - Конвертирует ВИДЕО в MP4 формат. 
- Output filename template : Шаблон именования файлов. Ээээ, я бы не трогал эту штуку, но как знаете, я не знаю как ее редактировать...
